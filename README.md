# 境外证券收益个税计算器

基于中国居民个人所得税规则，自动解析富途证券年度结单，计算境外证券投资的应纳税所得额。

## 功能特性

- **Excel 解析** — 直接上传富途年度账单和利息股息汇总 Excel 文件，自动识别并解析
- **多年联合分析** — 支持同时导入多年数据，跨年度追踪持仓成本（移动加权平均法）
- **股票交易** — 按标的分别计算已实现收益，支持港股 / 美股 / 多市场
- **基金收益** — 通过期初期末持仓 + 申购赎回流水计算净收益
- **分红利息** — 汇总各账户分红 / 利息收入，计算已代扣税额和应补缴税额
- **期权行权股票** — 单独追踪期权行权获得的股票，纳入对账但不重复计税
- **对账验证** — 自动校验计算结果与账户净值变化的一致性（含未实现盈亏）
- **年度对比** — 多年数据横向对比视图
- **报告导出** — 一键导出完整的中文个税计算报告（.txt）

### 所需文件

从富途牛牛导出以下文件（支持多年）：

| 文件类型 | 命名示例 |
|---------|---------|
| 年度账单 | `2024_年度账单_14621048.xlsx` |
| 利息股息汇总 | `2024_利息股息及其他收入汇总_14621048.xlsx` |

> 建议同时上传所有年份的文件，以确保跨年持仓的成本计算准确。

### 2025 年起：AEOI PDF 转换

自 2025 年度起，富途将「利息股息及其他收入汇总」改为「自动交换财务账户资料（AEOI）」PDF，
且不再随附参考汇率。本工具提供两种方式将其转换为可识别的标准 xlsx：

#### 方式一：页面内转换（推荐）

应用首页「AEOI PDF 转换工具」入口（默认折叠，点击展开）：上传 AEOI PDF，工具会在浏览器内解析出
年份、账户与股息/利息/其他收入，填入目标年度 12 月的 IRD 汇率后，一键生成并下载标准 xlsx。

> 浏览器直连 IRD 官网会被跨域(CORS)拦截，故汇率默认需手动填写——面板内提供对应年度的 IRD 汇率页面链接，
> 打开查看 `Dec <年份>` 的「USD→HKD」与「HK$100→人民币」两个数值填入即可。

#### 方式二：命令行脚本

使用以下脚本将该 PDF 转换为本工具可识别的标准 xlsx：

```bash
npm run convert:aeoi -- "富途年度结单/2025_自动交换财务账户资料_14621048.pdf"
```

脚本会在 PDF 同目录生成 `2025_利息股息及其他收入汇总_<牛牛号>.xlsx`，包含与历史年份一致的
三张工作表（账户信息 / 股息、利息及其他收入 / 参考汇率）。

- **汇率（自动抓取）**：PDF 不含汇率，脚本会**自动从香港税务局(IRD)官网抓取目标纳税年度「12 月」的官方平均汇率**，
  不沿用往年数据。IRD 按财政年度（4 月—次年 3 月）分页，目标年 12 月落在「该年/次年」财年页，
  URL 规则 `https://www.ird.gov.hk/eng/tax/bus_aer<NN>.htm`（`NN` =（目标年+1）的后两位，如 Dec 2025 → `bus_aer26.htm`）。
- **其它选项**：
  - `--out <xlsx>` 指定输出路径
  - `--year <年份>` 覆盖自动识别的年份
  - `--rate-html <html>` 离线/无网络时，从本地保存的 IRD 页面解析 Dec 汇率
  - `--rate-from <xlsx>` 直接从其它 xlsx 复制「参考汇率」表（最高优先级）
  - `--ird-url <url>` 覆盖默认 IRD 页面 URL

生成后将该 xlsx 与年度账单一起上传即可正常计算与对账。

## 计算方法

### 股票成本 — 移动加权平均法

每次买入后重新计算加权平均成本，卖出时按平均成本结转：

```
买入后平均成本 = (原持仓成本 + 本次买入金额) / (原持仓数量 + 本次买入数量)
卖出收益 = 卖出金额 - 卖出数量 × 平均成本 - 手续费
```

### 税率

| 所得类型 | 税率 | 说明 |
|---------|------|------|
| 财产转让所得 | 20% | 股票 / 基金已实现收益 |
| 股息红利所得 | 20% | 境外已代扣税额可抵免 |

### 汇率

使用富途年度结单中提供的参考汇率（年末汇率），支持 USD → HKD → CNY 链式转换。

### 对账公式

```
调整后总收益 = 已实现收益 + 未实现盈亏变动 + 期权股票收益
差额 = 调整后总收益 - (账户净值变化 - 外部净资金流入)
```
