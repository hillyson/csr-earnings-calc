# Scout Report — CSR Earnings Calculator

## 项目概况
- 全新项目，无已有代码
- 需求：纯前端 Web 工具，解析富途年度结单 Excel 文件，计算年度应纳税所得额
- 数据来源：2021-2024 年富途年度结单（xlsx 格式）

## 数据结构分析

### 年度账单（_年度账单_*.xlsx）
每个文件包含以下 Sheet：
1. **账户信息** — 姓名、牛牛号、账户号码、账户名称、年份
2. **证券-持仓总览** — 期初/期末持仓（代码、数量、价格、币种）
3. **证券-资金总览** — 期初/期末账户资金余额
4. **证券-交易流水** — 买入卖出记录（时间、代码、方向、数量、价格、成交金额、手续费）
5. **证券-资产进出** — IPO 获配、期权行权、股票代码变更、账户升级迁移
6. **证券-资金进出** — IPO 申购/退款、分红、手续费、利息等

### 利息股息及其他收入汇总（_利息股息及其他收入汇总_*.xlsx）
1. **账户信息** — 同上
2. **股息、利息及其他收入** — 每个账户的年度股息、利息、其他收入（含币种）
3. **参考汇率** — 年末各币种对港币汇率

## 关键业务场景

### 需要计算的资产类型
1. **股票（证券）** — 港股 + 美股，盈亏可互抵
2. **基金** — 单独计算，不与股票互抵
3. **分红/利息** — 单独统计

### 成本计算特殊情况
- IPO 获配：从资金进出推算（申购金额 - 退款金额 + 手续费 + 融资利息）
- 红股：票面价计算成本（如 HKD 1/股）
- 期权行权（LIZI/SOGP）：已完税，排除计算
- 股票代码变更（LIZI → SOGP）：需关联处理
- 账户升级迁移：持仓不变，仅账户变动

### 跨年持仓追踪
- 2022 年买入腾讯 700，分多次买入，2024 年分次卖出
- 需要移动加权平均成本法追踪

## 技术决策
- 纯前端实现，无后端
- 需要 Layer 4 UI E2E 测试
- 前端根目录：项目根目录
